## Getting Started

First, run the development server:

■ FrontEnd 진행순서<br>

1 - npm i (필요한 파일 설치)<br>

2 - cd .\FE\my-app (my-app폴더까지 이동<br>

3 - npm next build (배포 및 설치)<br>

4 - npm start (실행)<br>

5 - http://localhost:80/login으로 이동후 Login
